Quick Setup (5 Minutes)

1

Install on Your Development Machine

brew install tailscale

For Linux: curl -fsSL https://tailscale.com/install.sh | sh

2

Install on Your Mobile Device

üì± App Store ü§ñ Google Play
3

Sign In on Both Devices

Use the same account (Google, Microsoft, or GitHub). They'll automatically connect to each other.

4

Start AI Maestro

cd ai-maestro
yarn dev

Server starts on port 23000

5

Access from Mobile

Open your mobile browser and navigate to:

http://YOUR-MACHINE-NAME:23000

Find your machine name in Tailscale app ‚Üí Machines

üí° Pro Tip

Bookmark the URL on your mobile device for instant access. Add it to your home screen for a native app-like experience!
