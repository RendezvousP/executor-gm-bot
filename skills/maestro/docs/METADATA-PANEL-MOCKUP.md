- Sidebar: 280px (fixed)
- Main: flex-1
- Metadata Panel: 400px (default, resizable 280-600px)

### Laptop (1024-1439px)
- Sidebar: 280px (fixed)
- Main: flex-1

- Main: 100%
- Metadata Panel: Bottom sheet (slide up, 60vh)

### Mobile Portrait (<768px)
- Sidebar: Full-screen overlay
- Main: 100%
- Metadata Panel: Bottom sheet (slide up, 70vh)
